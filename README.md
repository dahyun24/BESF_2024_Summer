# KyungHee University BESF laboratory projects
A collection of projects done in the BESF lab <br><br>



## 1. 인공지능 등을 활용한 한약(생약) 관능검사 보조기술 개발 연구 
기간 : 2023.03 ~ 2024.05

역할 및 주요 성과 : 
- PyQt5를 활용한 사용자 친화적인 GUI 개발
  : 실시간으로 한약재 이미지를 촬영하면 인공지능 모델이 해당 이미지를 분석하여 결과를 직관적으로 시각화하는 인터페이스를 설계 및 구현하였습니다.
  
- 한약재 분류 모델 연동
  : 이미지 처리 및 분류 모델을 PyQt5 기반 GUI와 성공적으로 통합하여, 사용자가 실시간으로 한약재 이미지를 판별할 수 있는 시스템을 개발하였습니다.
  
- 프로토타입에서 실사용 단계까지의 개발
  : 초기 프로토타입 단계부터 시작하여, 기능적 완성도와 사용성 테스트를 통해 실사용 가능한 안정적인 소프트웨어로 발전시켰습니다. 이를 통해 연구팀의 한약재 관능검사 보조 기술 연구에 중요한 기여를 하였습니다.

## 2. 스마트 온실용 개방형 지능 제어시스템 개발 
기간 : 2024.06 ~ 현재

역할 및 주요 성과 :
- BOSCH CtrlX 기반 스마트 제어 시스템 구현
  : BOSCH 사의 CtrlX 제어기를 중심으로, 스마트 온실 환경에서의 효율적인 제어 시스템을 설계하고 구현하였습니다. 센서 노드와 구동기 노드 간의 데이터 교환을 원활하게 하기 위해, 통신 프로토콜 설정 및 제어 시스템 통합을 담당했습니다.

- KS3267 표준 준수 및 RS485 Modbus 통신 구현
: 프로젝트의 요구사항에 따라 KS3267 표준을 엄격히 준수하면서, RS485 Modbus 통신 프로토콜을 성공적으로 구현하였습니다. 이를 통해 다양한 센서 및 구동 장치와의 안정적이고 신뢰성 높은 데이터 교환을 가능하게 하였습니다.

- Node-RED를 활용한 통신 및 제어 로직 개발
: Node-RED를 사용하여 센서 데이터 수집 등 IoT 통신 흐름을 시각화하였습니다. 이를 통해 스마트 온실의 환경 제어를 실시간으로 모니터링하고 최적화할 수 있는 시스템을 구축하였습니다.
